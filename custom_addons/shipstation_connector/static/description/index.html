SDLC Maintenance QR Code

Welcome to the official documentation for SDLC Maintenance QR Code — a module for Odoo 17, 18 & 19 Community Edition that generates unique equipment codes, creates scannable QR codes for maintenance equipment, and prints professional QR label sheets in multiple layouts.

What is Maintenance QR Code? #

This module extends Odoo's standard Maintenance module by adding auto-generated equipment codes, scannable QR codes that link directly to equipment records, and a print wizard for producing professional label sheets. Warehouse and maintenance teams can scan QR labels on physical equipment to instantly pull up the corresponding record in Odoo — no manual searching required.

Key Features #

🔟

Auto Equipment Codes

Unique codes generated automatically on creation using a yearly-resetting sequence (MEQ/2026/00001).

▩

Scannable QR Codes

QR codes encode a direct URL to the equipment form. Scan with any mobile device to open the record.

🖨

Label Printing

Print professional QR label sheets in Dymo, 2x7, 4x7, or custom layouts with zero-margin PDF output.

🎨

Custom Header Color

Choose the label header color via a color picker to match your company branding.

📋

Multi-Copy Printing

Print multiple copies of each label in a single batch for placing on equipment and spare tracking.

🔒

Manager-Only Code Generation

Equipment code generation restricted to Equipment Managers for controlled asset tracking.

Quick Start #

  1. Install the module — Download and install sdlc_maintenance_qr_code from the Odoo Apps menu.
  2. Open Maintenance — Navigate to Maintenance > Equipment.
  3. Create or open equipment — Equipment codes are auto-assigned on creation. For existing equipment, click Generate Code.
  4. View the QR code — Open the QR Code tab on the equipment form to see the live QR preview.
  5. Print labels — Click Equipment QR Label, choose layout and options, then click Print Labels.

Getting Started

Install the module and start generating QR labels for your maintenance equipment with zero configuration.

Prerequisites #

  • A working Odoo 17, 18, or 19 Community Edition instance
  • The Maintenance (maintenance) module installed and configured
  • Admin or Equipment Manager access

Installation Steps #

  1. Download the Module
    Download sdlc_maintenance_qr_code from the Odoo App Store or obtain it from SDLC Corp.
  2. Upload to Your Server
    Place the folder in your Odoo addons directory.
  3. Update Apps List
    Go to Settings > Technical > Update Apps List and click Update.
  4. Install the Module
    Navigate to Apps, search for "SDLC Maintenance QR Code", and click Install.

Verify Installation #

After installation you will see:

  • A new Equipment QR Label button on the equipment form and list view header
  • A new Equipment Code field and Generate Code button on the equipment form
  • A new QR Code tab on the equipment form showing the live QR preview
  • An Equipment Code column in the equipment list view
  • Equipment Code searchable in the search bar
Equipment form showing the Equipment QR Label button, Equipment Code, and Generate Code link
ℹ No Configuration Needed

The module works out of the box. Equipment codes are auto-assigned on creation and the QR code sequence is set up automatically during installation.

Security & Access Control

The module uses Odoo's built-in Maintenance security groups with appropriate access levels for managers and regular users.

Code Generation Permissions #

The Generate Code button on the equipment form is restricted:

  • Only visible to users in the Equipment Manager group (maintenance.group_equipment_manager)
  • If a non-manager user somehow triggers the action, a UserError is raised
  • The button is hidden automatically once an equipment code has been assigned
ℹ Auto-Generation

Equipment codes are auto-assigned when new equipment records are created, regardless of the user's group. The Generate Code button is only needed for equipment that existed before the module was installed.

Equipment Code Sequence
Docs›Configuration›Equipment Code Sequence

Equipment Code Sequence

Equipment codes are generated automatically using an ir.sequence with a yearly-resetting counter.

Code Format #

MEQ/%(year)s/#####
PartDescriptionExample
MEQFixed prefix (Maintenance Equipment)MEQ
%(year)sCurrent year (resets counter each year)2026
#####5-digit zero-padded counter00001

Example Codes

  • MEQ/2026/00001 — First equipment created in 2026
  • MEQ/2026/00042 — 42nd equipment in 2026
  • MEQ/2027/00001 — First equipment in 2027 (counter resets)

Sequence Properties #

PropertyValue
Sequence Codemaintenance.equipment.code
PrefixMEQ/%(year)s/
Padding5 digits
CompanyGlobal (not company-specific)
No UpdateTrue (not reset on module reinstall)

Uniqueness Constraint #

A SQL constraint enforces that every equipment code is unique across the entire database:

⚠ Constraint

UNIQUE(x_equipment_code) — "Equipment code must be unique!" Attempting to assign a duplicate code will raise a database error.

Equipment Codes

Every piece of maintenance equipment receives a unique, auto-generated code that drives QR code creation and label printing.

Automatic Assignment (New Equipment) #

When you create a new equipment record, the module automatically assigns an equipment code using the maintenance.equipment.code sequence. No action required.

Manual Assignment (Existing Equipment) #

For equipment that existed before the module was installed:

  1. Open the equipment record from Maintenance > Equipment
  2. Locate the Equipment Code field (below Serial Number)
  3. Click the Generate Code link button next to the empty field
  4. The code is assigned instantly and the QR code is generated

QR Code Preview
Docs›Usage Guide›QR Code Preview

QR Code Preview

Each equipment record with a code displays a live QR code preview in a dedicated tab on the form view.

QR Code Tab #

Open any equipment record and navigate to the QR Code tab. The tab shows:

  • QR Payload (URL) — The full URL encoded in the QR code, displayed as a clickable link
  • QR Code Image — A 220x220 pixel live QR code rendered from the payload

QR Payload Format #

The QR code encodes a direct deep-link to the equipment form:

{base_url}/web#id={record_id}&model=maintenance.equipment&view_type=form

For example: https://myodoo.com/web#id=42&model=maintenance.equipment&view_type=form

Computed Fields #

FieldTypeDescription
x_qr_payloadChar (computed)Full URL that the QR code encodes
x_qr_urlChar (computed)URL to Odoo's barcode endpoint that generates the QR image
x_qr_image_htmlHtml (computed)HTML <img> tag displaying the QR code
ℹ When QR is Generated

QR fields are computed automatically whenever the x_equipment_code or company_id changes. If no equipment code exists, all QR fields are set to empty.

Print Wizard

The Print Equipment QR Labels wizard lets you configure and generate professional QR label sheets as PDF.

Opening the Wizard #

You can open the wizard in two ways:

From the Equipment Form

  1. Open an equipment record
  2. Click the Equipment QR Label button in the header
  3. The wizard opens pre-filled with the current equipment

From the Equipment List (Batch)

  1. Go to Maintenance > Equipment list view
  2. Select multiple equipment using checkboxes
  3. Click the Equipment QR Label button in the list header
  4. The wizard opens pre-filled with all selected equipment

QR Code Size Logic #

The QR code pixel size adapts automatically based on the layout:

ConditionQR Size

Pagination #

Labels are automatically distributed across multiple pages. If you have 20 equipment with 2 copies each (40 labels total) and use a 2x7 layout (14 per page), the report generates 3 pages.

📝 Page Breaks

Each page uses page-break-after: always in the PDF to ensure clean page separation. The PDF uses zero-margin A4 portrait paper format for printer-friendly output.

Scanning QR Codes
Docs›Usage Guide›Scanning QR Codes

Scanning QR Codes

Scan printed QR labels with any mobile device to instantly access the equipment record in Odoo.

Scanning Workflow #

  1. Locate the QR label on the physical equipment
  2. Open your camera app or any QR scanner on your mobile device
  3. Point at the QR code — the device reads the encoded URL
  4. Tap the link — your browser opens directly to the equipment form in Odoo
  5. View or update the equipment details, log maintenance requests, etc.

What the QR Code Contains #

The QR code encodes a deep-link URL that takes you straight to the equipment form view:

https://your-odoo.com/web#id=42&model=maintenance.equipment&view_type=form
⚠ Authentication Required

The user scanning the QR code must be logged into Odoo in their browser. If not logged in, they will be redirected to the Odoo login page first, then forwarded to the equipment record after authentication.

Use Cases #

  • Maintenance technicians — Scan equipment labels on the shop floor to quickly view maintenance history and log new requests
  • Inventory audits — Scan labels during physical inventory checks to verify equipment records
  • Onboarding — New team members can scan equipment to learn about each asset
  • Asset tracking — Track physical location by scanning labels during facility walks

Label Anatomy

Each printed QR label consists of four distinct sections designed for quick identification and scanning.

Label Structure #

PROPERTY OF ACME Corp
[QR Code]
MEQ/2026/00003
CNC Machine #7
SectionDescription
1. Header BandColored banner with "PROPERTY OF {company name}" in white uppercase text (8pt bold)
2. QR CodeScannable QR code (260–320px) encoding the equipment URL
3. Equipment CodeMonospace bold text (9pt) showing the code (e.g., MEQ/2026/00003)
4. Equipment NameEquipment name in 8pt text, truncated with ellipsis if too long

Label Styling Details #

ElementStyle

Cell Padding4px
Text AlignmentCenter
Header FontArial, 8pt bold, white, uppercase
Code FontCourier New (monospace), 9pt bold
Name FontArial, 8pt
OverflowHidden with text-overflow: ellipsis

Font Handling #

The module includes a SCSS stylesheet (report_qr_font.scss) loaded in the report assets bundle:

  • Label cells: Arial, Helvetica, sans-serif
  • Equipment code: Courier New, Courier, monospace (via .qr-code-mono class)
ℹ Special Character Handling

The module includes built-in mojibake repair (_repair_mojibake) and HTML entity encoding (_qr_display_html_entities) to handle equipment names with special characters, accented letters, or corrupted UTF-8 text in PDF rendering.

Customization
Docs›Label Design›Customization

Customization

Customize label appearance and layout to match your company branding and printing setup.

Header Color #

The header band color can be changed per print job via the color picker in the wizard:

SettingDefaultEffect
Header Color#6b4096 (purple)Background color of the "PROPERTY OF..." banner on each label

Use this to match your company brand colors. The text color on the header is always white for contrast.

Custom Grid Layout #

Select the Custom layout option in the wizard to specify any number of columns and rows:

  • Columns: Number of labels across the page (default: 2)
  • Rows: Number of labels down the page (default: 7)

This allows you to match any custom label sheet dimensions.

Multi-Copy Printing #

Set the Copies field to print multiple labels for the same equipment. For example:

  • Copies = 1: One label per equipment (default)
  • Copies = 2: Two labels per equipment (one for the asset, one for the file)
  • Copies = 3: Three labels per equipment (spare copies for damaged labels)

Paper Format #

PropertyValue

DPI96

💡 Printing Tip

When printing the PDF, make sure to set your printer to "Actual size" (not "Fit to page") to ensure labels align correctly with your label sheets.

QWeb Templates #

Two QWeb templates work together:

TemplatePurpose
report_simple_qr_label_cellRenders a single label cell (header + QR + code + name)
report_maintenance_equipment_qrMain report: iterates pages > rows > cells, calling the cell template

Rendering Flow

  1. Report triggered from wizard via action_print()
  2. _get_report_values() builds pages from equipment list, copies, and grid dimensions
  3. Main template iterates through pages with page breaks between each
  4. Cell template renders each label with the QR barcode widget, header, code, and name
  5. PDF generated with zero margins for label-sheet compatibility
